# Storage-BuosDilos-s

Um projeto para armazenar e organizar imagens usadas no site da equipe. Este repositório funciona como um storage de fotos e elementos visuais, permitindo manter os arquivos de mídia centralizados e acessíveis para desenvolvimento e publicação.

## Objetivo

- Centralizar imagens do site em um único repositório
- Facilitar a manutenção e atualização dos arquivos de mídia
- Garantir uma estrutura organizada para fotos, ícones e outros recursos visuais

## Conteúdo

O repositório deve conter:

- Fotos e imagens usadas nas páginas do site
- Ícones e elementos gráficos
- Pastas organizadas por tipo ou seção do site

## Como usar

1. Adicione novas imagens na pasta apropriada.
2. Nomeie os arquivos de forma clara e consistente.
3. Atualize qualquer referência no site para apontar ao novo arquivo, se necessário.

## Estrutura recomendada

- `images/` — imagens do projeto
  - `2026/` — exemplo de pasta para imagens de um ano ou seção específica

## Boas práticas

- Use nomes de arquivo descritivos e sem espaços
- Prefira formatos comuns como `.jpg`, `.png` e `.webp`
- Mantenha a organização das pastas para facilitar a busca e o versionamento

## Observações

Este repositório não é a aplicação em si, mas sim um armazenamento de mídia para o site. Ele deve ser usado apenas para arquivos estáticos e de imagem.
